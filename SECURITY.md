# 安全政策

## 支持的版本

| 版本 | 是否支持 |
|------|---------|
| v1.1 | :white_check_mark: |
| web-v1.0 | :white_check_mark: |
| < v1.0 | :x: |

## 报告漏洞

如果您发现了安全漏洞，请通过以下方式报告：

- 发送邮件至项目维护者
- 或通过 GitHub Issues 提交（如非敏感问题）

请勿在公开 Issues 中披露安全漏洞。

## 响应时间

我们会在收到报告后的 7 个工作日内进行初步回复，并尽力在 30 天内修复已确认的漏洞。

## 致谢

感谢所有帮助提高本项目安全性的贡献者。
